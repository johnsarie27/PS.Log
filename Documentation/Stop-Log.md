# Stop-Log

## SYNOPSIS
Stops log

## SYNTAX

```
Stop-Log [-Path] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Stops log with final message

## EXAMPLES

### EXAMPLE 1
```
Stop-Log -Path C:\temp\log.log
Adds the information-level log entry 'End logging'
```
